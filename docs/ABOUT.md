## What is Kaya?

Kaya is a modern, elegant application for playing and studying the ancient game of Go (also known as Baduk or Weiqi). Built with cutting-edge web technologies, Kaya combines a beautiful interface with powerful features for both beginners and experienced players.

## Features

- **Full Go Rules** - Complete implementation with proper capture and ko detection
- **Game Tree Navigation** - Explore game variations with an intuitive visual tree
- **SGF Support** - Import and export games in the standard Smart Game Format
- **Library Management** - Organize your games in folders
- **AI Analysis** - Real-time win rate, score estimation, and move suggestions via KataGo (ONNX)
- **Analysis Graph** - Visual win rate chart across all moves
- **Score Estimation** - Interactive territory and dead stone marking
- **Modern Interface** - Clean, distraction-free design with dark/light themes
- **Gamepad Support** - Navigate games with a controller
- **Cross-Platform** - Available as desktop app (Windows, macOS, Linux) and web app

## Technology

Kaya is built with:

- **React 18** - Modern UI framework
- **TypeScript** - Type-safe code
- **Tauri v2** - Lightweight desktop framework (Rust)
- **Rsbuild** - Fast build system
- **Bun** - High-performance JavaScript runtime

The core Go libraries are TypeScript ports from the excellent [Sabaki](https://github.com/SabakiHQ/Sabaki) project.
